Keyboard Shortcuts

These are shortcuts for the Mac programs we use in the Mobile Apps class.

Key Command Application
⌘-Space Spotlight Safari
⌘-S Save Safari
⌘-R Refresh Safari
⌘-T New Tab Safari/Windows
⌘-X Cut Safari/windows
⌘-C Copy Safari/Windows
⌘-V Paste Safari/Windows
⌘-a Select all Safari
⌘-shift-t Open last closed tab Safari
⌘-M Minimize Window Safari
